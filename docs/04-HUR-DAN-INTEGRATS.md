# 04 — ХУР / ДАН / Оператор холболт

> Энэ нь TARA-гаас **хамгийн их өөрчлөгдөх** давхарга — Эстонийн SK/X-Road-ийн оронд
> Монголын үндэсний дэд бүтэц.

---

## 4.1. Эстони ↔ Монгол дэд бүтцийн дүйцэл

| Үүрэг | Эстони (TARA) | Монгол (Гэрэгэ) |
|-------|---------------|------------------|
| Дата солилцооны давхарга | **X-Road** | **ХУР** (X-Road суурьтай) ✅ |
| Нэвтрэлтийн нэгдсэн систем | TARA / GovSSO | **ДАН** систем (одоо байгаа) |
| Иргэний бүртгэл | Rahvastikuregister | **УБЕГ** иргэний бүртгэл |
| Mobile-ID PKI | SK ID Solutions | Үүрэн операторууд (Mobicom/Unitel/G-Mobile) |
| CA / OCSP | SK гэрчилгээ | Монголын Үндэсний дата төв / CA |

> 💡 **Гол давуу тал:** ХУР нь Эстонийн X-Road технологид суурилсан тул TARA-гийн
> `xroad-client` загвар бараг **шууд тохирно** — зөвхөн security server хаяг, гишүүн
> ID, үйлчилгээний кодыг солих.

---

## 4.2. ДАН-тай харьцах стратеги

Монголд аль хэдийн **ДАН** (Дижитал Аналог Нэвтрэлт) систем байгаа. Гэрэгэ үүнтэй
хэрхэн зэрэгцэх вэ? Гурван хувилбар:

1. **Орлуулагч (replace):** Гэрэгэ нь ДАН-г бүрэн орлоно — урт хугацааны зорилт.
2. **Дээд давхарга (wrap):** Гэрэгэ нь ДАН-г **нэг адаптер** болгон дотроо дуудна
   (`auth-dan`), бусад аргыг шинээр нэмнэ. **Хамгийн бага эрсдэлтэй.**
3. **Туршилт (pilot):** Зөвхөн шинэ үйлчилгээнүүд Гэрэгэ ашиглаж, хуучин нь ДАН дээр
   үлдэнэ.

> ⭐ **Зөвлөмж:** Эхлээд хувилбар 2 (wrap) — ДАН-г дотроо адаптер болгож аваад,
> Гэрэгэ-гийн нэмэлт давуу талуудыг (биометр, орчин үеийн OIDC, SSO) дээр нь барина.

---

## 4.3. ХУР клиентийн урсгал (жишээ)

```
Гэрэгэ-Login                ХУР Security Server          УБЕГ үйлчилгээ
     │  иргэний РД-аар нэр асуух    │                          │
     │ ──────────────────────────▶ │  X-Road message          │
     │                             │ ───────────────────────▶ │
     │                             │ ◀── нэр, төрсөн огноо ─── │
     │ ◀── баталгаажсан мэдээлэл ── │                          │
```

Энэ мэдээлэл нь Hydra-д `accept login` болж буцах **claim**-уудыг бүрдүүлнэ:
`sub` (РД-ийн hash), `given_name`, `family_name`, `loa`.

---

## 4.4. Хууль зүйн нийцэл (Монгол)

- **Хүний хувийн мэдээлэл хамгаалах тухай хууль** — нэвтрэлтийн лог, иргэний дугаарын
  хадгалалт, зөвшөөрөл
- **Цахим гарын үсгийн тухай хууль** — гарын үсгийн хүчин төгөлдөр байдал
- **Кибер аюулгүй байдлын тухай хууль** — аудит, мониторинг шаардлага

> ⚠️ Эдгээрийг эхний үе шатнаас эрх зүйчтэй зөвлөлдөж тодорхойлох нь зүйтэй.
